Greeter
=======

A small library for formatting greetings for named recipients.

Usage
-----

Configure via :doc:`config` and call :func:`format_greeting`.
See the :class:`Greeter` class and `the guide <guide.rst>`_ for more,
or `an external page <https://example.com>`_ (not a cross-reference).
